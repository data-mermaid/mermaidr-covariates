# Search available covariates

Search available covariates

## Usage

``` r
search_covariates(title, description, as_data_frame = TRUE)
```

## Arguments

- title:

  Search term within title. Case insensitive.

- description:

  Search term within description. Case insensitive.

- as_data_frame:

  Whether to return the covariates as a data frame. Defaults to `TRUE`.

## Examples

``` r
# search_covariates("MEOW")
```
